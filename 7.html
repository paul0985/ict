What are the basic elements of a communication system?

The basic elements include the medium that carries the message, the sender who creates and sends the message, and the receiver who receives the message.

What does the term transmission mode refer to?

Transmission mode refers to the transmission of information between two communication devices via an interaction channel that indicates the direction of information flow.

What are the three primary types of transmission modes?

The three primary types are simplex, half-duplex, and full-duplex.

What is Simplex transmission mode?

Simplex mode is used for one-way flow of information from sender to receiver, such as in radio and TV transmissions.

What is Half-duplex transmission mode?

Half-duplex mode allows for two-way flow of information, but only one direction at a time, such as in walkie-talkies.

What is Full-duplex transmission mode?

Full-duplex mode allows simultaneous information flow in both directions, as seen in telephone networks.

What is Serial data transmission?

Serial transmission is where bits travel the same path, one after the other.

What is Parallel data transmission?

Parallel transmission is when a group of bits are transmitted at one time, each taking a separate path.

What is Synchronous transmission?

Synchronous transmission organizes data into groups or blocks, transferred at regular intervals, ensuring both devices know when data can be sent and received.

What is Asynchronous transmission?

Asynchronous transmission sends data when ready, using start and stop bits to identify bytes, making it less efficient than synchronous transmission.

What is Isochronous transmission?

Isochronous transmission ensures that data is delivered at specific times, such as when transmitting video and audio data together.

What is the OSI Model?

The OSI Model is a conceptual framework with seven layers designed to standardize and understand network communication.

What are the seven layers of the OSI Model?

The seven layers are: Physical, Data Link, Network, Transport, Session, Presentation, and Application.

What does the Physical Layer do?

The Physical Layer handles the actual physical connection and transmission of raw binary data over a physical medium.

What is the role of the Data Link Layer?

The Data Link Layer is responsible for node-to-node delivery, ensuring error-free data transfer between adjacent nodes.

What does the Network Layer do?

The Network Layer manages logical addressing, routing, and delivery of data packets across different networks.

What is the function of the Transport Layer?

The Transport Layer ensures reliable data transmission, managing segmentation, reassembly, and error checking.

What does the Session Layer manage?

The Session Layer is responsible for establishing, managing, and terminating sessions between applications on different devices.

What is the purpose of the Presentation Layer?

The Presentation Layer translates data between the application layer and the network, managing encryption and formatting.

What does the Application Layer do?

The Application Layer provides the interface for user applications to communicate over the network.

What is Circuit-Switching in data transmission?

Circuit-switching establishes a dedicated path for data transmission between sender and receiver, used in traditional telephone systems.

What is Packet-Switching?

Packet-switching separates messages into packets, which travel independently across the network and are reassembled at the destination.

What is Broadcasting in data transmission?

Broadcasting sends data packets to all nodes on a network, retrieving only by the intended recipient, primarily used in LANs.

What is a Protocol in communication?

A protocol is a set of rules and regulations that allow electronic devices to connect and exchange data.

What is TCP/IP?

TCP/IP stands for Transmission Control Protocol/Internet Protocol, enabling data communication on the internet by breaking data into packets.

What is FTP?

FTP stands for File Transfer Protocol, used for exchanging files between users on the Internet.

What is HTTP?

HTTP stands for Hyper Text Transfer Protocol, used for communication between web browsers and servers.

What is a URL?

A URL (Uniform Resource Locator) is the unique address of a web page or file on the Internet.

What is Ethernet?

Ethernet is the most widely used standard for wired networks, typically used with LANs that have a star topology.

What is Wi-Fi?

Wi-Fi is a family of wireless networking standards used for medium-range wireless communication.

What is IP Addressing?

IP Addressing assigns a unique numerical identifier to each computer on the Internet, formatted as four numbers separated by dots.

What is DNS Addressing?

DNS (Domain Name System) assigns a human-readable name to an IP address, making it easier for users to access websites.

What are the differences between Synchronous and Asynchronous transmission?

Synchronous transmission sends data in blocks at constant intervals, while asynchronous transmission sends data in bytes without synchronization.
